# Published Magazine Site

This folder is the local source tree for a future GitHub Pages (or similar static hosting) deployment.

Planned public structure:
- `/hn/YYYY-MM-DD.html`
- `/newsblur/YYYY-MM-DD.html`
- `/latest-hn.html`
- `/latest-newsblur.html`

Typical next steps:
1. create a GitHub repo for this folder
2. connect a custom subdomain like `mag.yourdomain.com`
3. point DNS at GitHub Pages
4. push this folder contents to the repo root (or configured Pages publishing branch)
